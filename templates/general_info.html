VECTr: Visual Encodings for Clonal Trees

How to use VECTR

VECTr is a web based tool for visualizing clonal trees. You can begin by selecting the trees you will be visualizing. We currently support NEWICK and DOT tree formats so any input tree will have to be in either format. There are two example trees that are set by default but to replace those you can either select a file or input the tree manually through the textbox. You can then select the file format and save your input.

Parent-child is the default distance metric for the tool

Input Formats

Newick

In mathematics, Newick tree format (or Newick notation or New Hampshire tree format) is a way of representing graph-theoretical trees with edge lengths using parentheses, curly brackets and commas. Some example newick trees are: 

            (,,(,));                        no nodes are named 
            (A,B,(C,D));                    leaf nodes are named 
            (A,B,(C,D)E)F;                  all nodes are named
